# task

## Help

```
> task
Usage: task <command> <args>

Commands:
   add <description> - adds task
   list              - displays task list
   remove            - removes a task
```

## Add

```
> task add one
> task add two
> task add three
```

## List

```
> task list
one
two
three
```

## Remove task

```
task remove one
```
